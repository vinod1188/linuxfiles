Absolute path
=============
It is nothing but complete path. always it will start with '/'.

Relative path
=============
It is nothing but continution path of pwd. it will never start with '/'.


# cp <source file path> <destination path>

# cp /vLearn/vmware/f1  /vLearn/UNIX
     -----------------
            |
       ABSOLUTE PATH

# cp f1 /vLearn/UNIX
     --
      |
  RELATIVE PATH



# cp -r /vLearn/vmware/dir1  /vLearn/UNIX

# cp -r dir1  /vLearn/UNIX


# mv /vLearn/vmware/f2  /vLearn/UNIX


# mv f2  /vLearn/UNIX

# mv -r /vLearn/vmware/dir1  /vLearn/UNIX

# mv -r dir1  /vLearn/UNIX
